# 🏥 Hospital Management System

A relational database project developed using SQL to simulate a Hospital Management System. The project focuses on designing an efficient database structure to manage hospital operations and maintain data integrity.

## 📌 Project Overview

This project includes the complete design and implementation of a hospital database, covering the main entities and their relationships.

## ✨ Features

- Database Requirement Analysis
- Entity Relationship Diagram (ERD)
- Relational Schema Design
- Database Normalization
- Table Creation using DDL
- Data Insertion using DML
- SQL Queries for Data Retrieval
- Primary Key & Foreign Key Relationships

## 🗂 Database Modules

- Department
- Staff
- Doctors
- Patients
- Nurses
- Appointments
- Prescriptions
- Pharmacy
- Laboratory
- Invoice
